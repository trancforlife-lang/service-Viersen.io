## Installation

### Lokal
1. Dateien entpacken
2. `index.html` im Browser öffnen

### Mit Python Server
```bash
python -m http.server 8000
```
Dann öffnen: http://localhost:8000

### Hosting
- Netlify: Drag & Drop Upload
- Vercel: `vercel deploy`
- GitHub Pages: Zu Repository pushen
- Traditionell: Per FTP hochladen

## Struktur
```
it-repair-shop-v2/
├── index.html          # Hauptseite
├── css/
│   └── style.css       # Alle Styles
├── js/
│   └── main.js         # JavaScript
├── assets/
│   └── images/         # Bilder (Platzhalter)
├── robots.txt          # SEO
├── sitemap.xml         # SEO
├── .htaccess           # Apache Config
└── README.md           # Diese Datei
```

## Anpassung
- Adresse/Kontakt in `index.html` ändern
- Farben in `css/style.css` unter `:root`
- Logo/Bilder in `assets/images/`
